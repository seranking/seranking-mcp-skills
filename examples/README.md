# Examples

Real, end-to-end runs of the skills in this plugin against publicly observable targets. Each example mirrors the exact `Output format` from the corresponding `SKILL.md` so you can preview what the skill produces before installing it.

## How to read an example

Each folder follows the convention `{skill-slug}-{target-slug}-{YYYYMMDD}/`. Inside:

- Numbered raw-data files (`01-*.md`, `02-*.md`, …) are the per-step outputs of the skill's `Process` section. They exist for auditability — you can trace any number in `REPORT.md` back to its source.
- `REPORT.md` is the synthesised deliverable, the artifact a human reviewer or stakeholder consumes directly.

## Snapshot disclaimer

These outputs are **single-point-in-time snapshots**. The SE Ranking AI Search prompt database refreshes monthly; share-of-voice numbers, prompt samples, and competitor positioning all drift. Treat the examples as illustrative of the *shape* of the deliverable, not as a statement about the brands today. Date stamps on every folder make freshness obvious.
